API Integration within Client

Our API has been integrated with this client to demonstrate its functionality and possible usages. The data is retrieved and visualized with the following JavaScript libraries:

jQuery
We've written a JavaScript function which is called upon submission of the HTML form. The API endpoint is then called using getJSON() method from jQuery's library with each request arguments appropriately set with inputs from the form. The JSON is then displayed to the user with colour coded syntax done by adding CSS dynamically with JavaScript

Chart.js
We display the JSON data retrieved in a form of a line graph with the assistance of Chart.js library. The data requires conversion to an array type, so for each query variable, an array was generated. Each array was then placed into its respective Chart() which populated the graph with data. The graphs title and axis labels were handled upon creation of each Chart()
