API Integration within Client

This is a brief description of how we integrated Envision API into this application. To see a demo application, go to the Query Generator.

Retrieving data from the API

We obtain the data by making a request to the Envision API. Since it is a RESTful service, we could use several programming languages to do this. Some examples using cURL, Python, and jQuery are available on the Documentation page. Since our demo application is web-based, we have used jQuery.

$.getJSON(`http://envision-api.ap-southeast-2.elasticbeanstalk.com/api/v1.0?instrument_id=ABP.AX&date_of_interest=2012-12-10&lower_window=3&upper_window=5&list_of_var=Return`, (data) => {
    let dataString = JSON.stringify(data, null, 3);
})

Once we have the dataString, we can display it with highlighted done by adding CSS dynamically with JavaScript.

Visualizing the data with Chart.js

For graphing, we have used Chart.js, although there are many other suitable charting libraries available in JavaScript too.

To display a graph we need to manipulate the data retrieved to the required format of the library. This is done with the code below.

The format of data from the Envision API can be obtained from the Documentation page. Our demo application is another good way of exploring the output of the API.

function buildGraphData(dates, datasets, graphOptions) {
    var formatOutput = {
        type: 'line',
            data: {
                labels: dates,
                datasets: datasets
            },
            options: graphOptions
        }
    return formatOutput;
}
